> Node module to Encode and Decode ArcticFox Config Files (.afc)

[ArcticFox](https://github.com/maelstrom2001/ArcticFox) is a Custom Firmware for Joyetech, Wismec and Eleaf battery mods 
by https://nfeteam.org/


## Usage

see [test.js](test.js)


## Related
